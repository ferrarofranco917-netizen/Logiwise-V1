# Kedrix One

Kedrix One è una piattaforma SaaS modulare enterprise costruita in HTML, CSS e Vanilla JavaScript con deploy su GitHub Pages.

## Stato corrente
- base grafica approvata preservata
- sidebar dinamica STEP 4A
- routing hash-based per moduli e sottomoduli
- placeholder coerenti per architettura enterprise
- workspace pratiche e anagrafiche base mantenuti

## Stack
- HTML
- CSS
- Vanilla JS
- PWA
- GitHub Pages root deploy
